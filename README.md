## AlgoFS
### Wut
Treat algorand like s3
### Y
find the price of storage in this network
### how
jam ~8k into a stateful application, then do filesystem things until someone gets mad.
### Done already:
* ~8k block read/write
### Want:
* multiblock objects
* multiobject storage
